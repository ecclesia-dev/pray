# pray

Catholic prayer CLI for the terminal. Traditional prayers only.

## Prayers

- **Rosary** — Joyful, Sorrowful, Glorious mysteries (traditional assignment, no luminous)
- **Angelus** / **Regina Cæli**
- **Morning Offering** with Guardian Angel and St. Michael
- **Night Prayers** with Examination of Conscience and Act of Contrition
- **Grace** before and after meals
- **Memorare**, **St. Michael**, **Act of Contrition**, **Apostles' Creed**

## Usage

```
pray                    # Rosary (today's mysteries)
pray rosary sorrowful   # Specific mysteries
pray angelus            # The Angelus
pray morning            # Morning Offering
pray night              # Night Prayers
pray grace              # Grace Before Meals
pray thanks             # Grace After Meals
pray list               # Show all prayers
```

## Install

```
make install
```

## Requirements

POSIX shell. No dependencies.
